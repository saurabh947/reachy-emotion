Reachy Emotion

Two apps in one: a continuous emotion-detection loop that plays matching robot animations, and a Gemini-powered voice conversation where emotion detection is a live tool call.

👁 Camera emotion detection 🎙 Microphone audio fusion 🔊 Spoken announcements 🎭 Recorded emotion moves 🤖 Gemini conversation 🗣 Voice input (STT)

Emotion loop: Detects emotions in real-time using Reachy's camera and microphone, plays matching pre-recorded animations, and announces the emotion through the speaker.


Gemini conversation: Talk to Reachy via voice. Gemini maintains the conversation history and calls the local emotion SDK as a tool when it wants to know how you're feeling.

What's inside

A multimodal emotion-response layer for your Reachy Mini.

👁

Visual detection

Processes frames from Reachy's built-in camera to detect faces and read facial expressions.

🎙

Audio fusion

Captures audio from the robot's microphone array and fuses it with visual cues for richer emotion detection.

🎭

Expressive reactions

Plays pre-recorded emotion animations from the Reachy Mini emotions library in response to what it detects.

🔊

Spoken announcements

Uses text-to-speech (gTTS) to announce the detected emotion through the robot's speaker.

👀

Face gaze tracking

Reachy's head automatically follows the detected face, making the interaction feel natural and attentive.

⚡

ActionCommand pipeline

Bridges the emotion detection SDK and Reachy Mini SDK via a clean ActionCommand interface.

🤖

Gemini conversation

Talk to Reachy via voice. Gemini maintains the full conversation history and decides when to call detect_emotion as a tool.

🗣

Voice input (STT)

Records speech from Reachy's mic with energy-based VAD, transcribes via Google Speech Recognition, and feeds text to Gemini.

How it works

Two apps, one install.

🎭 Emotion loop

  • 📸 Grabs a frame from Reachy's built-in camera
  • 🎙 Samples audio from the microphone array
  • 🧠 Runs face + audio fusion via EmotionDetector
  • 📦 Emits an ActionCommand → plays move + announces emotion

🤖 Gemini conversation

  • 🗣 Listens to your voice via Reachy's mic (VAD)
  • ✍️ Transcribes speech → sends to Gemini with full history
  • 🔍 Gemini calls detect_emotion tool when it wants to check your mood
  • 🔊 Plays Gemini's response as speech + matching robot move

Robot response actions

Both modes use the same Reachy Mini action vocabulary. The emotion result drives the choice of pre-recorded animation.

idle acknowledge comfort speak gesture stub

Installation

One command from your Reachy Mini dashboard, or install manually.

🖥 Dashboard (one-click)

Open Reachy Mini Control in your browser, find Reachy Emotion in the app store, and hit Install. All Python dependencies (including the emotion detection SDK) are fetched automatically.

🛠 Manual install

Clone this Space and install with pip or uv:

pip install -e . ffmpeg required for TTS
